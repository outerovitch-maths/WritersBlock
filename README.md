# Writer’s Block

This project is a **fork of a simple, distraction-free writing tool** designed to be lightweight, fast, and cross-platform friendly.

The goal is to keep writing as minimal and fluid as possible, while gradually evolving into a more powerful but still distraction-free editor.

---

## ✨ Features

- Minimal, distraction-free writing interface
- Dark mode support
- Adjustable font size
- Monospace / proportional font toggle
- Basic file open / save system
- Save As modal
- Local preferences stored in browser (theme, font mode)

---

## 🚧 Roadmap

Planned improvements:

### ⌨️ Emacs-like keybindings
- `Ctrl + X Ctrl + S` → Save
- `Ctrl + X Ctrl + C` → Exit / close session (optional)
- Additional modal-style command handling (Emacs-inspired workflow)

### 💾 Save system improvements
- True **Save / Save As distinction**
- Remember last saved file location (where supported by browser)
- Improve filename handling and extensions

### ⏱ Autosave system
- Automatic saving every X minutes
- Configurable interval
- Option to enable/disable autosave

### 🌐 Localhost Python companion
A small Python server to:
- Serve the editor locally
- Allow real file system persistence (true save/open from disk)
- Optional API for autosave sync
- Lightweight local “Writer’s Block Desktop mode”
